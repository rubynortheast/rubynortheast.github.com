Tuesday 27th November 2012 6.30pm for 7pm in Newcastle upon Tyne

Zero to Cloud in 40 minutes

Experienced, North East based, Linux System Administrator and open source advocate, Andrew Taylor (@ukandrewtaylor) will be talking all things Cloud at Ruby North East on Tuesday. Coming off the back of a very well received talk on Background Job Processing with Beanstalkd at PHP North East we are delighted to secure him for this month’s event. Andrew was recently snapped up by local FTSE 100 software giants, Sage (UK) Ltd., to help them in their global expansion into cloud hosted SaaS applications using Ruby and the Rails framework.

Previous talks have focused specifically on specialised Ruby cloud vendors such as Heroku and EngineYard, but Andrew will be approaching this topic from a System Administrator’s perspective. This makes it relevant to developers from any language or framework interested in hosting their applications in the cloud. This is a technical talk and will cover:

- What is the cloud?
- Cloud types and vendors compared
- Amazon Web Services as an example of available features
- Application design and architecture concerns

Register for the event on Eventbrite. Everyone is welcome. Room sponsored by Transcendit.

Tuesday 29th May 2012

Object Oriented Design in Ruby (with Rails)
Rails is the victim of its own success. As applications grow larger and more complex, the promise of "sustainable productivity" evapourates. Learn how the Rails community is embracing Object Oriented Design to better architect maintainable software. Data-Context-Interaction (DCI) and SOLID will be presented as complements to Model-View-Controller (MVC). Although discussing Ruby and Rails, the principles will apply to other frameworks and languages.

All are welcome. Philip Poots and Stephen Lloyd-Hirst present on Tuesday 29th May at 7pm at The Old Post Office, Pink Lane, Newcastle upon Tyne. Register with Eventbrite.

We have secured Pizza for this month's event (available from 6.30pm) thanks to sponsorship from Sage (UK) Ltd. They are currently on the lookout for a Ruby Developer.
